Chatbot Tester

Riverside County Auditor-Controller — internal admin tool

How to test

The chat widget is embedded in the bottom-right corner of this page — same widget production claimants would see embedded on auditorcontroller.org.

Click the chat bubble to open it, then try messages like "My name is Carey Ministries" to exercise the address-quiz flow, or "I'd like to talk to a person" to exercise the live-agent handoff.

This tester talks to the same WebSocket the production widget uses, so any chat session you start here is visible to admins on the dashboard's Chat handoffs page (when you trigger a handoff).
